# Alpha Course Website

A single-page site for your church’s Alpha course.

## Run locally

**Option 1 — Open in browser**

- Double-click `index.html`, or  
- Right-click → Open With → your browser

**Option 2 — Local server (recommended)**

From the project folder:

```bash
# Python 3
python3 -m http.server 8000

# Or npx (Node)
npx serve .
```

Then open **http://localhost:8000** in your browser.

## Contents

- **Hero** — “Explore life, faith & meaning” and primary sign-up CTA  
- **What is Alpha** — Short intro to Alpha  
- **What to expect** — Food, talk, discussion  
- **Stories** — Two embedded testimonial videos (YouTube)  
- **Join us** — Sign up button linking to your Google Form
